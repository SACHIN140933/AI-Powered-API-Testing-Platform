# AI-Powered API Testing & Security Analysis Platform

## Overview

An intelligent API testing platform built using FastAPI, Python, JavaScript, HTML and CSS.

The platform allows developers and testers to:

- Test REST APIs
- Import Swagger/OpenAPI specifications
- Generate automated test cases
- Run security validation tests
- Analyze API responses
- Detect common API issues
- Generate PDF reports
- Store and manage API collections
- Use AI-powered root cause analysis

---

## Features

### API Testing
- GET and POST request support
- Response time measurement
- Status code validation

### Swagger Import
- Import OpenAPI specifications
- Automatically discover API endpoints

### Security Analysis
- Validation testing
- SQL Injection testing
- XSS testing
- Authentication analysis

### AI Assistant
- Root cause analysis
- Error diagnosis
- Suggested fixes

### Reporting
- PDF report export
- API health reports
